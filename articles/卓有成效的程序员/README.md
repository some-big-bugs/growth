# 加速

## 启动面板

存在启动面板中的应用越多，则你寻找有用程序越困难，每个用户可能装10-20个应用程序来辅助开发和日常工作，其中只有4-6个才是常用的，剩余的可能一天只用一次，或者一个月只有一次，不要让这类软件占用着宝贵的第一级启动面板

## 启动器

使用启动器可以快速的找到你想启动的应用或者某些文件，而不是自己的多级目录中一个一个查找。
常用的启动器：

windows: 自带地址栏、wox、火萤酱、Everything
macos: 自带启动器、alfred

wox 介绍：https://www.jianshu.com/p/8fd212f5c01f
火萤酱介绍：https://www.huochaipro.com/

# 关注

## 隔离策略

一天中注意力很难维持，毕竟饿了会犯困，饱了也会犯困，不饿不抱总想吃饭。

阻止视觉和听觉干扰会帮助你集中一些注意力。
阻止听觉干扰：比如带上耳机，即使你不听歌，这样一方面可以较少听觉干扰，一方面可以阻止其他人打扰你工作（注：不要听相声，就算听也要听相声皇后的，不要听有情绪的中文歌，这样更容易降低注意力）。
阻止视觉干扰：将一些不重要的软件的提示关掉，例如 Facebook、Taobao、京东、音乐软件、视频软件

## 创造安静时间

可以规定在什么时间段不能开会，这样就可以有目的的将这段时间集中编码，必将将思路打断，则需要很长的时间才能恢复回来
